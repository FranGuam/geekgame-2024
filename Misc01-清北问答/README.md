### 题面

小北问答是 PKU GeekGame 的经典题目，主要目的是通过问答题的形式检验选手**在互联网上查找信息**的能力。

但无论阁下如何应对，规则都是一样的：**答对一半题目**可以获得 Flag 1、**答对所有题目**可以获得 Flag 2。

### 提示

- 第 5 题：PulseAudio 的 `pactl get-sink-volume` 命令可以看当前音量的分贝值。当然你也可以看源码。
- 第 6 题：这个建筑物不在北大校内，但是和北大校内的一个建筑物存在某种联系。

### 解答

1. 贺清华大学建校100周年
   - 直接搜索
2. pku-lostangel.oss-cn-beijing.aliyuncs.com
   - 看源码
3. 12
   - 搜索键盘图片，自己数即可
4. 5.2.1
   - 看源码的`package-lock.json`
5. 28.6
   - 表示音量有三种方式：数值、百分比、分贝值，数值范围为0~65535，百分比与之线性对应，分贝值在100%时对应0dB，换算公式为`60log_10(x)`
6. 通州北关
   - 图片前景的“七星公馆”在北京搜索，在其附近查找是否有塔形建筑
